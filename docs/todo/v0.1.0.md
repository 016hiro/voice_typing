# v0.1.0 — 完成清单

> 本版本目标：跑通"按 Fn → 录音 → 本地 ASR → 可选 LLM → 注入"的最小可用闭环。

## 已完成

- [x] SwiftPM 项目骨架 + Makefile（build / run / install / clean）
- [x] LSUIElement Info.plist + 麦克风 entitlements + ad-hoc codesign
- [x] `AppState` (@MainActor ObservableObject) + 持久化（语言、LLM 配置）
- [x] 5 语言 enum + Whisper 语言代码映射，默认 zh-CN
- [x] 麦克风 + 辅助功能权限请求与状态轮询
- [x] `FnHotkeyMonitor`：CGEventTap 监听 flagsChanged，回调返回 nil 抑制 emoji
- [x] `AudioCapture`：AVAudioEngine + RMS 实时流 + 16 kHz mono float32 累积
- [x] `SpeechRecognizer` 协议（prepare / transcribe / cancel + state stream）
- [x] `WhisperKitRecognizer`：`openai_whisper-large-v3` + downloadBase 自动下载
- [x] `TextInjector` (@MainActor) + `InputSourceManager`：剪贴板快照、CJK IME 检测/切换/还原、合成 Cmd+V
- [x] `LLMConfig` + `LLMRefiner`：OpenAI 兼容 API + 保守纠错 prompt + 失败回落原文
- [x] `CapsuleWindow` (NSPanel + NSVisualEffectView .hudWindow + 形状蒙版)
- [x] `CapsuleView` (SwiftUI HStack：波形 + 文字)+ 弹性宽度 PreferenceKey
- [x] `Waveform5BarView`：真实 RMS 驱动，权重 + attack/release 包络 + 抖动
- [x] `StatusItemController`：动态菜单，根据 AppState 重建
- [x] `SettingsWindow`：API Base URL / API Key（眼睛 + 清空）/ Model / Test / Save
- [x] AppDelegate 编排 Fn → 录音 → 转录 → refine → 注入 全链路
- [x] 入场/退场动画（alpha 0.35 / 0.22 s）+ 宽度 spring 过渡（0.25 s）
- [x] 修复 6 个开发期 bug（详见 [devlog/v0.1.0.md](../devlog/v0.1.0.md)）

## 本版本不做（已显式排除）

- ❌ Intel Mac 支持（仅 arm64）
- ❌ 自动模型管理（多模型选择 / 删除 / 重下）
- ❌ 多设备同步设置
- ❌ 云 ASR 后端
- ❌ 自定义快捷键（仅 Fn）
- ❌ 长音频分段流式转录（一次性 utterance，60 s 上限）
