## 主线 #B5：Per-app 热词配置

> **状态**：scope 待 `/think` 拍板，本身阻塞 schema 决策。

- 现状：`ContextProfile` 只携带 `systemPromptSnippet`；dictionary `state.dictionary.entries` 是全局
- 候选 schema：
  - **(a)** `ContextProfile.dictionaryFilter: [UUID]?` 引用全局 dict 子集（白名单）。global dict 仍是 superset
  - **(b)** 完全独立 `PerAppDictionaryStore`，profile 自带 entries
  - **(c)** `ContextProfile.dictionaryMode: enum { all / whitelist / blacklist }` + 配套 `[UUID]`
- UI 改动：profile editor sheet 加多选列表，从全局 dict 列出 checkbox
- pipeline 触点：`stopRecording` 路径上 `GlossaryBuilder` 的 entries 输入按当前 profile 过滤
