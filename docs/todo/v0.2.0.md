# v0.2.0 — 可切换 ASR 后端

> 本版本目标：把 ASR 从"固定 Whisper large-v3"升级到"多模型可切换"，为未来随技术演进替换底座打好底盘。

## 设计决策（已确认）

- **默认后端**：Qwen3-ASR-1.7B —— 新安装直接用 Qwen，不保留 Whisper 兜底。
- **模型清单**（固定三档，不含 Whisper 小模型）：
  - Whisper large-v3（WhisperKit，v0.1.0 已有）
  - Qwen3-ASR-0.6B（MLX，为未来流式预留）
  - Qwen3-ASR-1.7B（MLX，新默认）
- **模型管理**：
  - 切换后端**不删除**旧模型（下次切回来零等待）
  - 提供显式的"删除"入口，由用户主动清理磁盘

## 核心任务

### 协议与后端

- [x] `SpeechRecognizer` 协议保持不变（已留口子），新增 `ASRBackend` enum
- [x] 新增 `QwenASRRecognizer` 实现，依赖 `soniqo/speech-swift` v0.0.9（原 `ivan-digital/qwen3-asr-swift` 的改名后仓库）
- [x] 模型目录分家：`models/whisperkit/`、`models/qwen-asr-0.6b/`、`models/qwen-asr-1.7b/`
- [x] `ModelStore` 扩展：`isDownloaded` / `sizeOnDisk` / `delete` / `availableBytes` / `migrateV010WhisperLayoutIfNeeded`
- [x] `RecognizerFactory` 根据后端构造对应 recognizer

### AppState / 持久化

- [x] `AppState.asrBackend: ASRBackend` 持久化到 UserDefaults
- [x] `AppDelegate.switchBackend / activateBackend` 运行时无缝切换（unload 旧 Qwen → 构造新 → prepare）
- [x] 启动时自动迁移 v0.1.0 的 WhisperKit 旧路径

### 菜单 UI

- [x] 菜单新增 **Model** 子菜单，3 项（Whisper large-v3 · Qwen 0.6B · Qwen 1.7B），每项显示 `name · size · status`
- [x] 菜单新增 **Manage Models…** 入口

### 设置窗口（Tabbed）

- [x] `TabView`：Models / LLM 两个标签
- [x] Models 标签：三行模型表格，每行 [Switch/Download] 按钮 + 🗑️ 删除按钮 + 实时下载进度
- [x] 删除按钮二次确认（删除当前后端时文案特别警告）
- [x] LLM 标签：保留 v0.1.0 的 API Base URL / Key / Model / Test / Save 全部表单
- [ ] 空间不足预检查（检查磁盘剩余 vs 模型大小）—— 留到后续版本

### 基础设施

- [x] Swift tools-version 5.9 → 6.0，macOS 14 → 15，bundle version 0.1.0 → 0.2.0
- [x] `RecognizerState.loading(progress:)` 接入 Qwen 真实百分比（WhisperKit 仍 indeterminate）
- [x] Language.swift 新增 `qwenName`（chinese/english/japanese/korean/chinese for zhTW）
- [x] v0.1.0 → v0.2.0 迁移路径：启动时检测 `models/argmaxinc/...` 存在时迁移到 `models/whisperkit/argmaxinc/...`

## 验证（端到端）

1. 全新安装 → 默认 Qwen-1.7B 自动下载 → 录音转中文长句
2. Model → 切换到 Whisper large-v3 → 再录一次确认切换生效
3. 切回 Qwen-1.7B → 零等待（文件已缓存）
4. Manage Models → 删除 Qwen-0.6B（未下载）按钮应灰掉；删除已下载模型需要二次确认
5. LLM Refinement 在三种后端下都能跑
6. 升级旧版本（v0.1.0 已有 large-v3 下载在旧路径）：自动迁移，切换到 Whisper 无需重下
7. 下载中断（关网）后重启：Qwen 包内部有 3 次重试+退避，正常恢复

## 本版本不做

- ❌ 流式转录（Qwen-0.6B 的 TTFT 优势这版用不上，留到做流式时）
- ❌ 云 ASR 后端（OpenAI Whisper API）
- ❌ 多 LLM 后端（只做 ASR）
- ❌ 自定义模型（用户传自己的 ASR 权重）
- ❌ 空间不足预检查（小功能，留到 v0.2.1）
- ❌ 无 Intel Mac 支持（仍仅 arm64）
- ❌ 无 macOS 14 支持（speech-swift 要求 macOS 15+，v0.1.0 用户如果还在 Sonoma 需要停留在 v0.1.x）

## 风险与未知

- `soniqo/speech-swift` 还在 `v0.0.x` 阶段，API 可能在 1.0 前破坏性变更 —— 已 pin 到 `from: "0.0.9"`
- Qwen-ASR 首次下载 ~1.4 GB，受网速影响
- 同时下载多个模型的行为未测试
