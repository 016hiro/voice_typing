# v0.6.0.3 — Sparkle 真升级链路验证（v0.6.0.2 → v0.6.0.3）

> **目的**：v0.6.0.1 那次 #72 验证因为 Sparkle helper 签名 bug 失败。v0.6.0.2 修了签名（手动安装 baseline）。这一版从 v0.6.0.2 走 Sparkle 自动升级，把整条链路真正跑通。

## 改动

只 bump `Resources/Info.plist`：

- `CFBundleShortVersionString` `0.6.0.2` → `0.6.0.3`
- `CFBundleVersion` `16` → `17`

零代码 / 零配置改动。

## 验证项

在装着 v0.6.0.2 的机器上：

1. 菜单 → "Check for Updates…" → Sparkle 弹 "v0.6.0.3 available"
2. 点 Install → 下载 40MB → EdDSA 验签
3. 替换 + relaunch
4. 重启后 Settings → About 显示 `0.6.0.3`
5. **Microphone + Accessibility 授权不丢**（cdhash 应跨版本稳定）
6. Fn 录音转写正常

跑通后回填 `docs/release-process.md` Known quirks 段，正式 close v0.6.0 系列 + 关 #72。
