# v0.6.0.1 — Sparkle 真升级链路验证

> **目的**：v0.6.0 scope doc 第 7 项明确写"没跑完真升级验证不算 v0.6.0 ship"。这是兑现项，不是新功能。

## 改动

只 bump `Resources/Info.plist`：

- `CFBundleShortVersionString` `0.6.0` → `0.6.0.1`
- `CFBundleVersion` `14` → `15`

零代码改动。

## 用途

跑通 v0.6.0 → v0.6.0.1 的端到端自更新链路：

1. Sparkle 在装着 v0.6.0 的机器上 detect 到新版（自动 24h check 或菜单 → Check for Updates）
2. EdDSA 签名验证通过
3. DMG 下载 + 替换 + 重启正常
4. 重启后 Settings → About 显示 `0.6.0.1`
5. TCC 授权（Microphone + Accessibility）跨升级保留
